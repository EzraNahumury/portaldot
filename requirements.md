Introduction
Portaldot Mini Hackathon Online Season 1 is a lightweight online hackathon for developers who want to quickly build a prototype, validate an idea, and showcase their work.

This event focuses on low-barrier participation, runnable MVPs, and clear demos. It does not require a complex competition format or a production-ready commercial product.

Eligibility
This hackathon is open to developers worldwide.

We welcome:

Rust / Ink! / Substrate developers
Web2 / Web3 developers
Builders with solid engineering skills who can build alone or in teams
Projects should meet the following requirements:

Built on Portaldot
Use POT as gas
Deliver a runnable MVP
Be demo-ready
Keep core contracts open source
Submit a GitHub repo, README, and demo video
Judging criteria
Portaldot Native Deployment: The project must be deployed based on the features of the Portaldot blockchain and use its native token for gas. This is a mandatory eligibility criterion and cannot be copied from other sources.

Demo Completion: The project must have a functioning MVP that is demonstrable.

Application Value: The project must have market potential.

Presentation Quality: The live demonstration should be clear, with smooth, logical flow and the key points clearly highlighted.

Community Voting: Community voting is only used for the Community Favorite Award and does not contribute to the main judging score.
